Лекция № 17 "Система растровой компьютерной графики Adobe Photoshop. Редактирование, текст, слои."

Темы, рассматриваемые в данной лекции:

  1. Способы выделения, редактирования и трансформации элементов изображения.
  2. Коррекция изображений.
  3. Работа с текстом.
  4. Многослойные изображения. Работа со слоями.

1  Средства выделения фрагментов изображения

Если в изображении выделен какой-либо фрагмент, то для редактирования доступен только это фрагмент.
Имеются три основных вида инструментов выделения:

  • выделение фрагментов правильной формы;
  • выделение фрагментов произвольной формы;
  • выделение по цвету.

1.1   Выделение фрагментов правильной формы

Для этого используется инструмент Прямоугольная область (Marquee). Разновидности этого инструмента:

  • Прямоугольник (Rectangular);
  • Эллипс (Elliptical);
  • Одиночная строка (Single Row) – строка из одного пикселя;
  • Одиночная колонка (Single Column) – колонка из одного пикселя.

При выделении прямоугольной и эллиптической области можно выбрать стиль выделения (Стиль, Style):

  • Нормальный (Normal) – произвольная область.
  • Фиксированный размер (Fixed Size) – область фиксированного размера (его можно указать);
  • Фиксированный коэффициент пропорции (Fixed Aspect Ratio) – фиксированное соотношение размеров.

Имеется также параметр Растушевать (Feather) – выделяется область с размытыми границами. Чем больше значение этого параметра, тем больше размытость границ.
Использование инструментов Прямоугольник и Эллипс с клавишей Shift позволяет получить выделить, соответственно, квадрат и круг.

1.2   Выделение фрагментов произвольной формы

Для этого используется инструмент Лассо следующих видов:
Лассо – области произвольной формы;
Многоугольное лассо (Polygonal Lasso) - многоугольник произвольной формы;
Магнитное лассо (Magnetic Lasso) - граница выделяемой области автоматически проводится по линии изменения цвета или яркости (граница прилипает к такой линии).
При использовании инструмента Магнитное лассо на границе выделяемой области автоматически расставляются связующие точки. Можно перемещать эти точки. Для замыкания выделенной области щелкнуть по начальной точке, или щелкнуть мышью при нажатой клавише Ctrl.
Параметры инструмента Магнитное лассо:

  • Ширина (Width) – ширина полосы в пикселях анализируемой для определения границы, которую нужно построить;
  • Контрастность края (Edge Contrast) – чувствительность, задаваемая в пределах 1-100%.

Если границы выделяемой области достаточно чёткие, то можно указать параметры Ширина и Контрастность края достаточно большими и обводить желаемую область приближённо.
Имеются также параметры:

  • Частота (Frequency) – частота расстановки связующих точек. Чем сложнее по форме область, которую нужно выделить, тем большим следует задавать этот параметр;
  • Растушевать (Feather) - аналогично соответствующему параметру для инструмента Прямоугольная область.

1.3   Выделение по цвету

Для этого используется инструмент Волшебная палочка (Magic Wand). При щелчке этим инструментом по какому-либо пикселю выделяется вся область данного цвета, прилегающая к выбранному пикселю, или область данного цвета по всему изображению.
Параметры инструмента Волшебная палочка:

  • флажок Непрерывный (Contiguous): если этот флажок установлен, то выделяется только область, прилегающая к месту щелчка;
  • флажок Использовать все слои (Use all layers): если он установлен, то выделяются области выбранного цвета во всех слоях изображения;
  • параметр Обратная чувствительность (Tolerance): значения от 0 до 255. Указывает количество выбираемых цветовых тонов, близких к выбранному. Если указано 0, то выбираются только области, в точности соответствующие выбранному цвету.

1.4   Общие указания по выделению фрагментов изображения

При использовании любых инструментов выделения или их комбинации можно задать на панели параметров один из четырех режимов выделения:

  • Новое выделение (New selection) - при выделении новой области прежнее выделение теряется;
  • Добавить к выделению (Add to selection) - новое выделение добавляется к прежнему;
  • Вычесть из выделения (Subtract from selection) - новое выделение исключается из прежнего;
  • Пересечь с выделением (Intersect with selection) - выбирается пересечение прежнего и нового выделения.

Отмена любого выделения – клавиши Ctrl+D или команда Выделение – Убрать выделение (Select – Deselect).
Иногда удобно выделять не изображение, а фон, например, по цвету (с помощью инструмента Волшебная палочка), а затем поменять выделенные области местами командой Выделение – Инвертировать выделение (Select – Inverse).

2.  Слои

2.1   Создание слоев. Основные операции со слоями

Изображение в Photoshop может состоять из одного или нескольких слоев. Все слои имеют одинаковый размер, цветовую модель, разрешение.
Новый файл состоит из одного слоя – Фон (Background). Для этого слоя при создании файла можно указать способ заполнения: белый цвет, прозрачный слой, цвет фона.
Для работы со слоями используется палитра Слои (Layers) или меню Слои. Можно создавать новые слои или удалять имеющиеся, используя соответствующие инструменты в палитре и меню Слои.
Можно создавать новый слой из части имеющегося слоя, т.е. скопировать или вырезать часть изображения в отдельный слой. Для этого выделить желаемую часть и использовать команду Слои – Новый – Слой через копирование или Слой через вырезание.
При создании нового слоя, по умолчанию, он полностью прозрачен, а изображение на нём - непрозрачно. Управление прозрачностью слоя рассматривается ниже (п.2.4.4).
Положение слоев относительно друг друга можно изменять с помощью мыши в палитре Слои, кроме слоя фона. Но если переименовать слой фона, то он становится обычным слоем, и его также можно перемещать.
В палитре Слои для каждого слоя (слева от рисунка, обозначающего слой) имеются два индикатора: индикатор видимости и индикатор состояния.
Если индикатор видимости содержит отметку в виде «глаза», то слой - видимый. Слой можно скрывать и снова делать видимым, щелкая мышью по этой отметке.
Если индикатор состояния (слева от отметки слоя) содержит отметку в виде кисти, это означает, что на слое можно рисовать. Отметка в виде цепи означает, что слой связан с другим слоем.
Разделение изображения на слои сохраняется только в форматах PSD и TIFF. При сохранении файла в других форматах все слои автоматически объединяются в один.

2.2   Совместная обработка слоёв. Связывание слоев

Иногда требуется выполнять перемещение, масштабирование, поворот и т.д. сразу для нескольких слоёв. Для этого удобно связать слои. В палитре Слои выбрать один из слоев и щёлкнуть по индикатору состояния другого слоя, с которым требуется связать выбранный слой (индикатор состояния слоя – поле в палитре Слои слева от изображения слоя). На индикаторе состояния при этом появится отметка в виде цепи. После этого, выбрав любой из связанных слоёв, можно выполнять их перемещение и другие операции сразу для всех этих слоёв. Для развязывания слоёв необходимо ещё раз щёлкнуть по индикатору состояния.

2.3   Защита слоёв

Для каждого слоя возможен один из четырёх режимов защиты:

  • Заблокировать прозрачные пиксели (Lock transparent pixels) – на таком слое можно изменять только уже имеющееся изображение, но не свободные пиксели;
  • Заблокировать пиксели изображения (Lock image pixels) – защита всех пикселей слоя;
  • Заблокировать позицию (Lock position) – запрет на операции перемещения;
  • Заблокировать все (Lock all) – полная защита слоя.

2.4   Стиль слоя

Стиль слоя – набор параметров слоя, определяющих его внешний вид и взаимодействие с другими слоями.
В палитре Слои имеются следующие средства управления стилем слоя:

  • Непрозрачность (Opacity) – настройка прозрачности слоя (100 – полностью непрозрачный слой);
  • Заливка (Fill) – аналогично настройке непрозрачности, но действует только на отдельные эффекты изображения;
  • меню режимов смешивания (настройка по умолчанию – Обычный), задающее взаимодействие слоя с нижележащими слоями.

Для настройки других параметров слоя используется команда Слой – Эффекты слоя (Layer – Layer Style).

2.5   Объединение слоёв

Несколько слоев можно объединить в один. Такое объединение позволяет уменьшить размеры файла, но теряется возможность обработки изображения по частям. Имеется несколько способов объединения слоёв:

  • связать слои, которые требуется объединить и воспользоваться командой Слой – Слияние связанных;
  • cделать видимыми только те слои, которые нужно объединить, и воспользоваться командой Слой – Слияние с видимым.

Кроме того, можно объединить все слои командой Слой - Выполнить сведение.
Следует также обратить внимание, что все слои автоматически объединяются в один при сохранении файла в любом формате, кроме PSD и TIFF.

2.6   Сборка многослойного изображения из нескольких файлов

Пусть требуется добавить изображение из файла_2 в файл_1 в качестве нового слоя. Это можно выполнить следующим образом.
1    Открыть оба файла.
2    Используя команду Изображение – Размер изображения, установить для обоих файлов одинаковые размеры изображения и одинаковое разрешение.
3    Перейти в файл_2. Скопировать в нём все изображение в буфер.
4    Перейти в файл_1. Создать в нём новый слой.
5    Скопировать на новый слой содержимое буфера.

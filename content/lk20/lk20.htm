3.7  Использование текста в CorelDRAW

3.7.1   Основные операции с текстом

Для ввода текста используется инструмент Текст. Текст в CorelDRAW может быть двух видов: художественный (Artistic) и простой (Paragraph).
Для ввода художественного текста выбрать инструмент Текст, щёлкнуть мышью в желаемом месте и вводить текст.
Для перехода в новую строку обязательно нажимать Enter. Для художественного текста нет понятия абзаца, поэтому к нему нельзя применять некоторые операции форматирования: нельзя установить красную строку, разместить текст в несколько колонок и т.д. Художественный текст следует выбирать, если к нему предполагается применять специальные эффекты, например, размещение по контуру, раскрашивание, изменение формы символов и т.д.
Для ввода простого текста требуется выбрать инструмент Текст, затем с помощью этого инструмента построить рамку и в ней вводить текст. Текст вводится в пределах этой рамки. Размер рамки можно затем изменить желаемым образом. К простому тексту можно применять обычные операции форматирования: выравнивание, оформление абзацев, размещение в несколько колонок и т.д.
Добавление к тексту новых символов и другие изменения могут выполняться двумя способами:

  • непосредственно с помощью инструмента Текст;
  • с помощью команды Текст – Редактировать текст (Text – Edit Text) или инструмента Редактировать текст в панели настройки.

Во втором случае появляется окно для редактирования текста. Этот способ удобен, если редактируемый текст имеет сложный вид, например, размещён вдоль кривой.
Форматирование текста выполняется с помощью инструментов панели настройки, команд из меню Текст, а также с помощью команды Текст – Форматировать текст (Text – Edit Text). В последнем случае появляется окно с набором инструментов форматирования.
Многие операции выполняются с текстом так же, как с любым объектом:

  • выделение текста и последующие операции, как с любым выделенным объектом;
  • выбор цвета для текста;
  • преобразование в набор кривых (после такого преобразования текст перестаёт быть текстом, а превращается просто в набор кривых).

Текст можно выделить также с помощью инструмента Форма (Shape), а затем перемещать отдельные символы текста и изменять расстояние между строками или между символами с помощью мыши: если при этом удерживать нажатой клавишу Ctrl, то изменяется интервал между символами, а при нажатой клавише Shift – между словами.

3.7.2   Построение текста по контуру

Ввод текста по контуру
Имеются два способа ввода текста по контуру (т.е. размещения текста вдоль некоторой кривой, границы объекта и т.д.):

  • выбрать инструмент Текст, подвести курсор к желаемому контуру. Курсор приобретает вид, соответствующий выбранной операции. Щёлкнуть по контуру в желаемом месте и ввести текст;
  • выбрать контур, затем выбрать команду Текст – Подогнать текст к пути (Text –Fit Text to Path). Курсор появится в начале выбранного контура. После этого можно ввести текст.

Текст, введенный по контуру – художественный текст.
Размещение имеющегося текста по контуру
Имеются два способа размещения текста по контуру:

  • выделить текст и контур и выбрать команду Текст – Подогнать текст к пути (Text – Fit Text to Path);
  • выделить текст, затем выбрать команду Текст – Подогнать текст к пути (Text – Fit Text to Path) и с помощью появившегося курсора выбрать желаемый контур.

Операции с текстом, размещенным по контуру
Контур и текст, размещённый вдоль него, представляют собой один объект. Можно выделять текст и контур по отдельности с помощью мыши при нажатой клавише Ctrl. После этого для них по отдельности можно выбирать цвет, а также по отдельности удалять.
Если требуется отделить текст от контура, то необходимо выделить их и выбрать команду Компоновать – Разбить текст вдоль кривой на части (Arrange – Break Text Apart). После этого текст и контур становятся разными, независимыми друг от друга объектами. Текст при этом сохраняет форму контура. Если требуется после этого выровнять текст (т.е. разместить его как обычный текст, без какой-либо связи с прежним контуром), то используется команда Текст – Выравнивание текста (Text – Straighten Text).
Для перемещения текста вдоль контура щёлкнуть по тексту и перетянуть за маркер красного цвета.
В панели настройки имеется набор инструментов для управления текстом, размещенным вдоль контура. Можно, например, изменять его расположение относительно контура (размещать текст над контуром, под контуром, поверх контура и т.д.).

3.7.3   Размещение текста внутри заданной области

Для этой операции имеются два основных способа:

  • ввод текста в контур;
  • применение к тексту оболочки (невидимой границы заданной формы).

Ввод текста в контур
1    Выбрать инструмент ввода текста.
2    Поместить его на желаемый контур (в который нужно ввести текст) и добиться чтобы курсор имел специальный вид.
3    Щелкнуть мышью и ввести текст.
Контур должен представлять собой набор кривых. Введенный текст будет простым. Текст и контур представляют собой один объект, но их можно выделять по отдельности с помощью мыши при нажатой клавише Ctrl. Чтобы разделить текст и контур: выделить их выбрать команду Компоновать – Разбить простой текст внутри контура (Arrange – Break Paragraph Text inside a Part Apart).
Применение оболочки к тексту
Для этого используется закреплённое окно Оболочка (Envelope). Для его вызова на экран выбрать команду Окно – Докеры (Window – Dockers). Далее возможны три случая:

  • применение к тексту одной из стандартных оболочек. Для этого:

1    Выбрать простой текст, к которому требуется применить оболочку.
2    В окне Оболочка (Envelope) нажать кнопку Ранний (Add preset).
3    Из появившегося набора в окне Оболочка (Envelope) выбрать желаемую оболочку и нажать кнопку Принять (Apply). Текст заключается в выбранную оболочку;

  • применение к тексту контура, построенного заранее:

1    В рабочем листе создать текст и контур.
2    В окне Оболочка (Envelope) нажать кнопку Создать из (Create From).
3    С помощью появившегося курсора выбрать желаемый контур и нажать кнопку Принять (Apply). Текст заключается в выбранный контур;

  • построение желаемого (произвольного) контура вокруг текста. Для этого выделить текст и в окне Оболочка (Envelope) нажать кнопку Новый (Add new). Вокруг текста появляется прямоугольный контур в виде набора кривых. Его можно отрегулировать как обычный набор кривых.

Чтобы отделить текст от оболочки, выделить его и выбрать команду Эффекты – Удаление оболочки (Effects – Clear Envelope).

3.7.4   Взаимное расположение текста и объекта

По умолчанию текст и другие объекты располагают поверх друг друга, как и все другие объекты.
Для управления взаимным расположением текста и объектов выделить объект и использовать кнопку Завернуть абзац текста (Wrap paragraph). В появившейся панели выбрать режим обтекания объекта текстом (см. рисунок 3.7).



Рисунок 3.7 – Режимы обтекания объекта текстом

Основные режимы обтекания объекта текстом:

  • Контур (Contour) – текст будет обтекать объект непосредственно по контуру объекта;
  • Квадрат (Square) – текст будет обтекать объект по прямоугольной рамке вокруг объекта.

В поле Text Wrap можно указать расстояние от объекта до текста.

3.7.5   Связывание текстовых блоков

Каждый фрагмент простого текста представляет собой текстовый блок.
Связанные текстовые блоки обладают следующим свойством: при заполнении одного из них текст автоматически продолжается в следующем связанном блоке. При удалении одного из связанных блоков текст из него не исчезает, а переходит в следующий связанный блок. Связанные блоки можно разместить как на одной странице, так и на нескольких.
Есть два основных способа создания цепочки связанных блоков:

  • создание нового блока, связанного с текущим. Для этого выделить блок, щёлкнуть по отметке в нижней стороне блока. С помощью появившегося курсора нарисовать на экране новый блок. Он будет связан с предыдущим. Новый блок можно создать и на другой странице;
  • связывание двух и более существующих блоков. Для этого щелкнуть по отметке в середине нижней стороны блока, появившимся курсором щелкнуть внутри другого курсора. Блоки свяжутся.

Примечание – Таким способом можно связывать текстовый блок не только с другим текстовым блоком, но и с любым замкнутым контуром.
Существующие текстовые блоки можно также связывать, выделив их и используя команду Текст – Связать (Text – Link).
Для разделения блоков имеются два основных способа:

  • выделить их и воспользоваться командной Текст – Развязать (Text – Unlink);
  • щелкнуть по отметке на нижней границе блока, затем щёлкнуть на пустом месте рабочего листа.

3.8  Работа с растровой графикой

3.8.1   Загрузка растровых изображений

В CorelDRAW можно загружать растровые изображения, например, подготовленные в Photoshop. Для этого используется команда Файл – Импорт. После того, как выбран файл с растровым изображением, можно выбрать его размещение на рабочем листе в CorelDRAW. Для этого существуют три возможности:

  • щёлкнуть по рабочему листу мышью. Изображение размещается в выбранном листе;
  • с помощью мыши построить на рабочем листе рамку. Растровое изображение размещается в рамке;
  • нажать Enter. Растровое изображение размещается в центре рабочего листа.

Кроме того, при импорте растровых изображений можно установить флажок Внешне связать растровое изображение (Link Bitmap Externally). В файл CorelDRAW вставляется не само изображение, а ссылка на него (т.е. на растровый файл). В этом случае уменьшается размер файла, но изображение нельзя будет редактировать средствами CorelDRAW.
Изображение, загруженное в виде ссылки, можно преобразовать в обычное растровое изображение, встроенное в CorelDRAW, командой Битовые изображения – Разорвать связь (Bitmap – Break Link).
При импорте растрового изображения имеются возможности загрузить его целиком, выполнить обрезку или изменить его размер.

3.8.2   Операции с растровым изображением как с объектом

Растровые изображения в CorelDRAW можно выделять, как и любой другой объект и выполнять с ним большинство операций, как с обычным объектом: выполнять поворот, растяжение и сжатие, наклон, зеркальное отображение и т.д. Поверх растрового изображения или за ним можно размещать любые другие объекты, в том числе текст. Порядком наложения объектов можно управлять с помощью команды Компоновать – Порядок (Arrange – Order). Растровые изображения можно группировать с любыми другими объектами командой Компоновать – Группа (Arrange – Group).
Форму растрового изображения можно изменять произвольным образом с помощью инструмента Форма (Shape). Данный инструмент позволяет изменять границу растрового изображения как любую кривую. При этом можно создавать новые узлы, изменять их типы и т.д. (см. п. 3.5.3). При этом растровое изображение не удаляется, а только становится невидимым (маскируется). Если затем с помощью данного инструмента восстановить прежнюю форму, то изображение снова становится видимым.
Некоторые операции для растровых изображений невозможны, например:

  • соединение объектов (см. п. 3.6.2);
  • формирование объекта из нескольких (п. 3.6.3).

Растровое изображение можно использовать как фон для объектов. Это позволяет получать растровые изображения сложной формы.
Пример – Пусть требуется получить изображение пятиконечной звезды с изображением головы орла внутри. Для этого выполнить следующее.
1    Импортировать файл Eagle, входящий в комплект файлов Photoshop.
2    Поверх импортированного растрового изображения поместить фигуру, например прямоугольник. Залить этот прямоугольник каким-либо цветом, например белым. При этом растровое изображение будет скрыто.
3    Поверх фигуры поместить звезду.
4    Выделить прямоугольник и звезду. Соединить их командой Компоновать – Комбинировать (Arrange – Combine), как показано в п. 3.6.2.
Пересечение соединённых объектов (в данном случае – прямоугольника и звезды) становится полностью прозрачным, и через него виден фон, в данном случае – орёл.
Таким образом, растровые изображения сложной формы можно получать как минимум двумя способами:

  • изменяя границы растрового изображения с помощью инструмента Форма (Shape);
  • путем наложения объектов на растровое изображение и их соединение.

Примечание – Еще один способ построения растровых изображений сложной формы (фигурная обрезка) приведен в п.3.9.2.

3.8.3   Редактирование растровых изображений в CorelDRAW

Для этого в составе пакета CorelDRAW имеется отдельная программа редактор растровой графики Corel Photo Paint. Чтобы вызвать её, требуется дважды щёлкнуть в CorelDRAW по растровому изображению, или выделить растровое изображения и выбрать команду Битовые изображения – Редактировать битовое изображение (Bitmap – Edit Bitmap).
Кроме того, отдельные операции с растровой графикой можно выполнять, используя меню Битовое изображение (Bitmaps), например:

  • Режим (Mode) – позволяет выбрать для растрового изображения цветовую модель;
  • Повторная выборка (Resample) – позволяет изменить размеры и разрешение изображения;
  • Конвертировать в битовое изображение (Convert to Bitmap) – позволяет изменить цветовую модель и разрешение, и т.д.

3.8.4   Преобразование растрового изображения в векторное

Выбрать растровое изображение и вызвать команду Битовое меню – Трассировать изображение (Bitmaps – Trace). Вызывается специальная программа преобразование растровой графики в векторную Corel Trace. В неё загружается выбранное изображение. Чтобы выполнить его преобразование, нажать кнопку Отследить (To Trace).
По окончании трассировки просто закрыть окно Corel Trace. Изображение будет представлять собой сгруппированный набор объектов. Чтобы теперь работать с ними по отдельности, требуется выделить изображение и выбрать команду Компоновать – Разгруппировать (Arrange – Ungroup All).
Пример – Пусть требуется для использования в CorelDRAW получить контур листа растения. Это можно выполнить следующим образом.
1    Загрузить файл растровой графики Водяные лилии (из папки Мои рисунки).
2    Выполнить его трассировку.
3    Выделить полученное векторное изображение и разгруппировать его.
4    С помощью мыши выбрать контур листа.

3.8.5   Преобразование векторных изображений в растровое

Эта операция требуется, если необходимо построить изображение в CorelDRAW, а затем использовать его в PhotoShop или другой программе растровой графики.
Чтобы преобразовать векторное изображение в растровое, требуется выделить его и выбрать команду Битовое изображение - Конвертировать в битовое изображение (Bitmaps – Convert to Bitmaps). Выбрать разрешение и цветовую модель, затем нажать ОК.
Чтобы сохранить полученное растровое изображение в одном из растровых форматов, требуется выбрать растровое изображение и воспользоваться командой Файл – Экспорт (File – Export), или щёлкнуть правой кнопкой на растровом изображении и выбрать Сохранить битовое изображение как (Save Bitmap As).

3.9  Эффекты

Большинство эффектов выполняется с помощью закреплённых окон, вызываемых на экран с помощью команды Эффекты (Effects) или команды Окно – Докеры (Window – Dockers). Имеется также несколько инструментов в стандартной панели инструментов, предназначенных для создания эффектов. Это инструменты, в названии которых есть слово Интерактивный.
Любой эффект можно отменить, выделив объект, к которому он применен, и выбрав команду Эффекты – Убрать эффект (Effects – Clear Effects).
В качестве примеров рассмотрим два эффекта: плавный переход от одного объекта к другому и фигурную обрезку объекта.

3.9.1   Эффект плавного перехода

Пример – Построим переход между прямоугольником и кругом.
1    Построить прямоугольник и круг.
2    Вызвать на экран закреплённое окно командой Эффекты – Смешанное (Effects – Blend).
3    Выделить два объекта, между которыми требуется выполнить плавный переход (в данном случае - прямоугольник и круг) и в закреплённом окне Плавный переход (Blend) нажать кнопку Принять (Apply).
Между выделенными объектами строится по прямой линии набор объектов постепенно меняющих форму от прямоугольника к кругу. Результат этой операции – сложный объект (группа объектов), включающая два исходных объекта и группу промежуточных объектов.
Если выделить полученный объект и применить команду Компоновать – Разбить группу с перетеканием (Break Blend Apart), то от группы отделяются исходные объекты. Промежуточные объекты при этом остаются одной группой. Если применить к ним команду Компоновать – Разгруппировать, то можно будет работать с каждым из промежуточных объектов отдельно.
При выполнении эффекта перехода в закреплённом окне имеется большой набор средств для управления эффектом.
Пример – Пусть требуется построить переход не по прямой, а по заданному пути (кривой). Для этого выполнить следующее.
1    Построить переход между объектами, как показано выше. Построить также кривую для желаемого пути перехода.
2    Выделить построенный переход (в данном примере – переход от прямоугольника к кругу). Затем в закреплённом окне эффекта щёлкнуть по кнопке Путь (Path). Из появившегося меню выбрать команду Новый путь (New Path).
3    Щёлкнуть появившемся курсором по кривой, вдоль которой должны располагаться промежуточные фигуры.

3.9.2   Фигурная обрезка

Для этой операции используются два объекта. Один из них называется Содержимое (Contents), а другой – Контейнер (Container). Контейнером может быть любая замкнутая кривая, а также художественный текст (Artistic Text). Содержимым может быть любой объект, чаще всего используют растровое изображение.
1    Разместить на рабочем листе содержимое и контейнер.
2    Выделить содержимое. Выбрать команду Эффекты – Power Clip – Разместить внутри контейнера (Effects – Power Clip – Place Inside Container).
3    С помощью курсора выбрать контейнер. Контейнер заполняется в соответствии с выбранным содержимым.

3.9.3   Копирование и клонирование эффектов

Эти операции позволяют применить эффект, уже примененный к какому-либо объекту, к другим объектам.
1    Выбрать объект, на который требуется скопировать эффект.
2    Выбрать команду Эффекты – Копировать эффекты или Клонировать эффекты (Effects – Copy Effects или Clone Effects).
3    Из появившегося субменю выбрать желаемый эффект.
4    С помощью появившегося курсора выбрать объект, с которого требуется скопировать эффект (причём нужно выбирать именно ту часть объекта, где имеется эффект).
Если выполнено клонирование, то при изменении эффекта у исходного объекта он изменяется также у клонированного объекта.

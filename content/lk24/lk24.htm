4.10 Визуализация

Визуализация – построение растровых изображений на основе созданных трёхмерных изображений. Визуализация необходима, чтобы видеть результаты применения материалов и источников света.

4.10.1 Простая визуализация

Простая визуализация – визуализация со стандартными параметрами. Чтобы выполнить ее, нажать Shift+Q или F9. Создаётся окно с трёхмерным изображением в соответствие с выбранным видом (Top, Front, Left, Perspective). Пример визуализации показан на рисунке 4.26.
В появившемся окне имеется также кнопка Save Image, позволяющее сохранить изображение в файл в желаемом растровом формате.

4.10.2 Настройка параметров визуализации

Для этого выбрать команду Rendering – Render. В появившемся окне перейти на вкладку Command. Основные настраиваемые параметры визуализации следующие:

  • вкладка Common:
  • Time output – выбор диапазона кадров для визуализации. Эта настройка требуется, если используется анимация. Для изображений без анимации выбрать настройку Single;
  • Output size – выбор размера изображения;
  • Viewport – выбор вида (слева, справа, перспектива и другие);
  • Render output – настройка вывода изображения в файл.
  • вкладка Raytracer – дополнительные параметры для изображений, содержащих прозрачные и отражающие материалы.

После настройки всех параметров нажать кнопку Render. Строится трёхмерное растровое изображение.

4.10.3 Изменение фона визуализации

Имеется возможность выбрать фон для изображения, которое строится при визуализации. Для этого требуется выбрать команду Rendering Environment. Открывается окно Environments and Effects. Далее имеются две основных возможности:

  • выбрать цвет фона. Для этого в области Background использовать поле Color;
  • выбрать в качестве фона любое растровое изображение. Для этого в области Background щёлкнуть по кнопке Environment Map. Затем в появившемся окне выбрать Bitmap, затем – выбрать желаемое растровое изображение. После этого закрыть окно Environments and Effects.

После настройки фона требуется выполнить визуализацию нажатием Shift+Q или F9.
На рисунке 4.27 показан пример визуализации изображения с примитивом Capsule (чтобы построить его, в командной панели выбрать вкладку Create – Geometry, выбрать набор примитивов Extended Primitives и воспользоваться кнопкой Capsule). В качестве фона выбрано растровое изображение (фотография холмистой местности).

4.11    Анимация

В 3ds Max имеются два основных вида анимации:

  • ключевая анимация;
  • анимация на основе движения объекта по заданному контуру.

4.11.1 Ключевая анимация

Принцип работы ключевой анимации следующий: пользователь задаёт параметры движущегося (анимируемого) объекта (положение, поворот, размер и т.д.) только в некоторых ключевых кадрах анимации. Для промежуточных кадров эти параметры определяются автоматически.
Элементы управления анимацией располагаются в нижней части экрана (см. рисунок 4.28).


Рисунок 4.28 – Элементы управления анимацией

Имеются два основных режима ключевой анимации:

  • автоматический (Auto Key);
  • принудительный (Set Key).

Пример (ключевая анимация в автоматическом режиме) – Получить изображение летающего волана для игры в бадминтон.
1    В окне Front построить примитив Cone. Он будет представлять собой изображение волана. Поместить его в исходную (левую или правую) точку его траектории.
2    Нажать кнопку Auto Key.
3    Установить таймер на 25-й кадр. Переместить волан в верхнюю точку его траектории.
4    Установить таймер на 48-й кадр. Разместить волан в крайней точке его траектории.
5    Установить таймер на 52-й кадр. Перевернуть волан.
6    Установить таймер на 75-й кадр. Переместить волан в верхнюю точку.
7    Установить таймер на 100-й кадр. Переместить волан в исходную точку.
8    Выключить кнопку Auto Key.
9    Чтобы просмотреть анимацию, нажать кнопку Play Animation.
10  Для остановки проигрывания анимации нажать Stop Animation.
В ходе настройки ключевой анимации в панели управления анимацией (см. рисунок 4.28) отображаются ключи, т.е. отметки изменения параметров:

  • красная отметка – изменение положения;
  • зелёная – поворот;
  • синяя – изменение размеров.

Эти ключи можно перемещать, удалять, редактировать.
Для настройки анимации в принудительном режиме нажать кнопку Set Key. Затем поочерёдно устанавливать объект в желаемые позиции аналогично тому, как это делается в автоматическом режиме, и в каждой из этих позиций устанавливать ключи с помощью кнопки Set Keys (кнопка с изображением ключа). Для выбора типа ключа используется кнопка Key Filters.
Для точной настройки траектории анимации используется окно Curve Editor, вызываемое командой Graph Editor – Track View – Curve Editor. Траекторию анимации можно редактировать как обычную кривую.

4.11.2 Анимация на основе движения по заданному контуру

Пример – Получить изображение шара, катающегося по окружности.
1    Создать шар и окружность, как обычные примитивы.
2    Выделить шар. В командной панели перейти на вкладку Hierarchy. Нажать кнопку Affect Pivot Only и поместить появившийся указатель на край шара. Выбранная точка шара будет использоваться в качестве опорной точки при его движении. Выключить кнопку Affect Pivot Only.
3    Указать, что шар будет двигаться по заданному пути. Для этого выделить шар и перейти в командной панели на вкладку Motion. На ней открыть вкладку Assign Controller. В ней выбрать поле Position. Нажать кнопку Assign Controller. В появившемся окне (см. рисунок 4.29) выбрать Path Constraint и нажать кнопку OK.



Рисунок 4.29 – Выбор режима движения по заданному контуру

4    Указать путь для движения. Для этого в области Path Parameters нажать кнопку Add Path. Щёлкнуть указателем мыши по окружности. Шар перемещается на неё.
5    Запустить анимацию нажатием кнопки Play animation.

Темы, рассматриваемые в данной лекции:

4.3  Модификаторы

Модификатор – операция, применяемая к объекту. К объекту можно применить несколько модификаторов. Всегда есть возможность изменить их порядок, а также отменить некоторые модификаторы.
Пример – Рассмотрим применение нескольких простых модификаторов, позволяющих сужать, растягивать, а также скручивать объект.
1    Построить объект Box (любым способом). Выделить его. В командной панели перейти на вкладку Modify, а на ней - в область Parameters (если она свернута, развернуть ее щелчком по знаку «плюс»). Ввести параметры объекта: Length Segs = 1, Width Segs = 1, Height Segs = 10 (это означает, что параллелепипед будет состоять из десяти сегментов, расположенных «в высоту»). Объект должен иметь примерно такой вид, как показано на рисунке 4.4 (показаны вид слева и перспектива).



Рисунок 4.4 – Объект Box, состоящий из десяти сегментов

Из списка модификаторов Modifier List выбрать желаемый модификатор, в данном случае – модификатор Stretch. Установить параметр выполняемой операции: Stretch = 0,2. 4.4  Построение трёхмерных объектов с помощью сплайнов

Сплайн (Spline) – плоская фигура. Сплайн может представлять собой стандартную плоскую фигуру (например, линия, круг, звезда) или иметь произвольную форму. Во многих случаях сплайны используются как основа для построения сложных трёхмерных объектов. Трёхмерный объект может строиться, например, путём вращения или перемещения сплайна.

4.4.1   Построение и редактирование сплайнов

Чтобы построить сплайн, выполнить следующее.
1    На командной панели выбрать вкладку Create – Shapes и выбрать желаемый сплайн (если требуется плоская фигура произвольной формы, выбрать Line).
2    Построить желаемый сплайн с помощью мыши.
Примечание – Сплайн-линия Line строится щелчками мыши. Каждый щелчок создаёт опорную точку, в 3ds Max называемую Vertex. Для окончания построения линии щёлкнуть правой кнопкой мыши.
Основные элементы (подобъекты) любого сплайна:

  • опорные точки (Vertex);
  • сегменты (Segment);
  • сплайн в целом (Spline).

Опорные точки аналогичны узлам в CorelDRAW (см. раздел 3.4). Они могут быть следующих типов:

  • Corner (угловая);
  • Smooth (гладкая);
  • Bezier и Bezier Corner – опорные точки с направляющими для управления их положением (аналогично узлам в CorelDRAW).

Для редактирования сплайна перейти на вкладку Modify, в стеке модификаторов перейти на уровень подобъектов, выбрать уровень Vertex. После этого можно, например, изменять типы опорных точек, перемещать их и т.д.
Для изменения типа опорной точки щелкнуть по ней правой кнопкой мыши и выбрать нужный тип из появившегося меню
Для перемещения опорных точек использовать инструмент Select and Move.
Основное средство редактирования сплайнов – модификатор Edit Spline. Чтобы воспользоваться этим модификатором, требуется выбрать сплайн, выбрать модификатор Edit Spline, перейти на уровень подобъектов и выбрать желаемую команду.
Рассмотрим, например, установку новой опорной точки (Vertex). Для этого выполнить следующее.
1    Выбрать сплайн, применить к нему модификатор Edit Spline, перейти на уровень подобъектов – опорных точек (Vertex).
2  В командной панели выбрать тип опорной точки.
3  Выбрать команду Insert.
4  С помощью появившегося инструмента щёлкнуть в желаемой точке сплайна.
5  Установить созданную опорную точку в желаемое положение.
6  Для окончания щёлкнуть правой кнопкой.
Некоторые другие команды модификатора Edit Spline:

  • Refine – установка новой опорной точки на сплайне аналогично команде Insert, но без перемещения.
  • Detach – отделение части сплайна;
  • Delete – удаление выделенной части сплайна;
  • Chamfer и Filler – придание углам в опорных точках правильной формы;
  • Mirror – зеркальная копия сплайна.

Модификатор Lathe

Для использования этого модификатора требуется построить сплайн, имеющий форму поперечного сечения половины желаемого объекта. Сплайн должен быть незамкнутым. Сплайн может иметь опорные точки любого вида, но две крайние опорные точки должны иметь тип Corner.

Для этого выполнить следующее.
1    Построить сплайн (типа Line), показанный на рисунке 4.10.

Рисунок 4.10 – Пример сплайна
2    Применить к нему модификатор Lathe.
3    В командной панели выбрать параметры модификатора Lathe, чтобы получить фигуру, аналогичную показанной на рисунке 4.9. Фигура строится вращением выбранного сплайна. Основные параметры модификатора Lathe, которые может потребоваться установить для решения данной задачи, следующие:

  • X, Y, Z – направление оси вращения;

Чтобы установить координаты опорной точки, требуется выделить ее и ввести координату в нижней панели.

4.4.3   Построение трёхмерных объектов путём выдавливания контура сплайна. Модификаторы Extrude, Bevel, Bevel Profile

Модификатор Extrude строит трёхмерную фигуру по форме заданного замкнутого сплайна. Основной параметр этого модификатора – Amount (толщина построенной фигуры).
Пример – Построить диск.
1    Построить сплайн Circle (т.е. круг).
2    Применить к нему модификатор Extrude. Задать параметр Amount. Результат см. на рисунке 4.11.



Рисунок 4.11 – Пример использования модификатора Extrude


Пример – Построить шестигранную гайку с круглым отверстием.
1    Перейти в режим построения сплайнов (Create – Shapes – Splines). Нажать кнопку NGon (т.е. выбрать построение сплайна-шестиугольника).
2    Нажать кнопку Keyboard Entry, чтобы построить выбранную фигуру (шестиугольник) путём указания её параметров (а не с помощью мыши). Задать параметры сплайна:

  • x = 0, y = 0, z = 0;
  • Radius = 60.

3    Аналогично, используя ввод параметров с клавиатуры, построить окружность (сплайн Circle) с параметрами:

  • x, y, z = 0;
  • Radius = 40.

Примечание – Построение сплайнов путем ввода параметров с клавиатуры (а не с помощью мыши) потребовалось, чтобы расположить сплайны относительно друг друга абсолютно точно, а именно – чтобы в точности совпадали центры окружности и шестиугольника.
4    Выделить шестиугольник и применить к нему модификатор Extrude, задав толщину, например, Amount = 5.
5    Выделить круг и к нему также применить модификатор Extrude, задав параметр Amount не меньше, чем у шестиугольника.
6    Выполнить булеву операцию вычитания (см. п. 4.2.7): из шестигранника вычесть цилиндр. Для этого выполнить следующее:
а) перейти на вкладку Create. Выбрать из меню Compound Objects;
б) выделить шестигранник;
в) нажать кнопку Boolean;
г) установить логическую операцию Substraction A-B;
д) нажать кнопку Pick Operand B и выбрать цилиндр. Выполняется вычитание. Результат должен быть аналогичен показанному на рисунке 4.12.

4.5  Построение объёмного текста

С текстом можно выполнять все операции, как с любым другим сплайном. Чтобы построить текст, выполнить следующее.
1    Перейти на вкладку Create. Выбрать на ней вкладку Shapes. Нажать кнопку Text.
2    В командной панели Parameters ввести текст. Указать параметры текста (шрифт, размер и т.д.).
Далее можно работать с текстом как с обычным сплайном, т.е. перемещать, копировать, применять модификаторы (например, Extrude и Bevel), выполнять булевы операции и т.д.

4.6  Построение трёхмерных объектов по заданным поперечным сечениям. Операция лофтинга

Лофтинг – построение поверхности по заданным поперечным сечениям, расположенным вдоль заданного пути. Поверхность (называемая «лофт») строится путём «натягивания» на заданную систему сечений.
Пример – Пусть требуется построить изображение трубопровода, построенного в виде прямоугольного контура (см. рисунок 4.15).


Рисунок 4.15 – Пример операции лофтинга

Здесь в качестве сечения будем использовать круг, а в качестве заданного пути – прямоугольник. Задача решается в следующем порядке.
1    Построить (например, в окне Top) сплайны, используемые в качестве сечения и пути, в данном случае - круг (сплайн Circle) и прямоугольник (сплайн Rectangle). Выделить один из них.
2    Перейти на вкладку Create. Выбрать на ней вкладку Geometry. Выбрать из меню в командной панели элемент Compound Objects. Нажать кнопку Loft.
3    Если на шаге 1 был выбран путь (т.е. прямоугольник), нажать кнопку Get Shape и выбрать сечение (т.е. круг). Если же, наоборот, на шаге 1 было выбрано сечение (т.е. круг), то нажать кнопку Get Path и выбрать путь (т.е. прямоугольник). Строится поверхность путем перемещения сечения вдоль заданного пути.

В операции лофтинга имеется возможность использовать в одной фигуре несколько сечений.
Пример – Пусть требуется построить изображение вазы с круглым дном и шестиугольным верхом (см. рисунок 4.17).
1    В окне Top построить круг (Circle) и шестиугольник (NGon). При этом шестиугольник должен быть больше, чем круг.
2    В окне Front построить вертикальную прямую линию (сплайн Line). Чтобы линия получилась в точности прямой, строить ее при нажатой клавише Shift.
3    Выполнить операцию лофтинга, как показано в предыдущем примере, используя в качестве пути линию, а в качестве сечения – круг. Будет построен вертикальный цилиндр (см. рисунок 4.16).


Рисунок 4.16 – Построение простого лофта

4    Выделить построенный цилиндр. Перейти на вкладку Modify. Убедиться, что в стеке модификаторов выбран модификатор Loft.
5    В поле Path установить желаемую позицию для применения нового сечения. По умолчанию она устанавливается в процентах от длины пути. Установить небольшую величину, например 20%.
6    Нажать кнопку Get Shape и выбрать новое сечение, в данном случае – шестиугольник. Построенный лофт изменяется, как показано на рисунке 4.17.

 


Рисунок 4.17 – Построение лофта с переменным сечением


Имеется также возможность изменять построенный лофт произвольным образом. Для этого открыть панель Deformation и нажать в ней кнопку Scale. Вызывается редактор Scale Deformation. В его окне имеется линия, соответствующая построенному лофту. На неё можно с помощью инструмента Insert Corner Point наносить новые точки, а затем перемещать их с помощью инструмента Move Corner Point, изменяя тем самым форму лофта.
Можно также изменять тип установленной точки. Для этого щёлкнуть по ней правой кнопкой и выбрать тип Corner, Bezier Corner или Bezier Smooth. По окончании настройки закрыть окно Scale Deformation.
Имеется также большой набор дополнительных операций, которые можно выполнять с построенным лофтом. Например, чтобы у полученного в примере изображения вазы был открытый верх требуется в панели Loft открыть панель Skin Parameters, снять флажок Cap Start (или Cap End).
Для придания вазе более реального вида, а именно – придания толщины стенок, следует применить к построенному лофту модификатор Shell и указать желаемую толщину стенок через параметры Inner Amount и Outer Amount.

4.7  Построение объектов с помощью каркасного моделирования. Модификатор Edit Mesh

Модификатор Edit Mesh позволяет работать с объектами на уровне их отдельных компонентов, а именно:

  • вершин (опорных точек, Vertex);
  • рёбер (Edge);
  • полигонов (Polygon);
  • граней (Face);
  • элементов (Element).

Общий порядок работы с модификатором Edit Mesh следующий.
1    Создать исходный объект, например примитив Box.
2    Применить к нему модификатор Edit Mesh.
3    В стеке модификаторов выбрать модификатор Edit Mesh. Перейти на уровень подобъектов и выбрать желаемый уровень.
4    Выбрать желаемый компонент, с которым нужно работать. Если требуется выбрать несколько компонентов, использовать клавишу Ctrl.
5    Выполнить желаемые операции, например, переместить выбранные компоненты с помощью кнопки Select and Move, или воспользоваться кнопками на вкладке Edit Geometry:

  • Create – создать;
  • Detach - отделить;
  • Extrude – выполнить выдавливание;
  • Bevel – создание фаски;
  • Chamfer – срез вершины или ребра;
  • Collapse – слияние, и многие другие.

Пример – Пусть требуется построить клин (см. рисунок 4.18).
1    Построить примитив Box.
2    Применить к нему модификатор Edit Mesh.
3    В стеке модификаторов перейти на уровень рёбер.
4    На одной из граней прямоугольника с помощью инструмента выделения при нажатой клавише Ctrl выбрать два параллельных ребра.
5    Нажать кнопку Collapse.

1    Построить примитив Box. При его создании задать количество сегментов:

  • Length Segs: 5;
  • Width Segs: 7;
  • Height Segs: 1.

2    Применить модификатор Edit Mesh.
3    В стеке модификаторов перейти на уровень Polygon.
4    C помощью мыши при нажатой клавише Ctrl выбрать четыре сегмента, где должны располагаться ножки стола.
5    В командной панели на вкладке Edit Geometry в поле Extrude ввести длину ножек, например, 40, и нажать кнопку Extrude.
6    Чтобы ножки стола получились суженными снизу, в поле Bevel указать величину сужения, например, -3, и нажать кнопку Bevel.

Пример – Получить фигуру, показанную на рисунке 4.20.

8    В стеке модификатор перейти на уровень Polygon.
9    Щелчком мыши выделить полученную поверхность.
10  Применить к ней операцию Extrude с параметром -5.

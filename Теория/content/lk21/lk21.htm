Лекция № 21 "Система трехмерной компьютерной графики 3ds Max. Двумерные и трехмерные фигуры."

Темы, рассматриваемые в данной лекции:

  1. Основные возможности 3ds Max. Настройка программного пакета и рабочей области. Работа с файлами.
  2. Построение двумерных и трехмерных фигур. Редактирование объектов.
  3. Плоскостное моделирование трехмерных объектов с помощью сплайнов: модификаторы сплайнов, выдавливание, поворот вокруг оси, лофтинг.

4.1  Начало работы с 3ds Max.
Построение простейших объектов

Рабочее окно программы 3ds Max показано на рисунке 4.1.


Рисунок 4.1 – Рабочее окно программы 3ds Max
Как видно из рисунка, стандартное рабочее окно состоит из трех окон проекций и окна перспективы (на рисунке 4.1 – правое нижнее окно). Справа располагаются командные панели, сверху – главная панель инструментов, снизу – панель инструментов для управления изображением.
Изображение можно строить в любом окне, но рекомендуется делать это в одном из окон проекций (а не в окне перспективы).
Построение простых объемных объектов
В 3ds Max имеется набор стандартных объемных объектов – примитивов (плоскость, параллелепипед, сфера, цилиндр и многие другие). Чтобы построить такой объект, сначала требуется выбрать его тип (например, параллелепипед, сфера и т.д.) Это можно сделать двумя способами:

  • выбрать элемент меню Create - Standard Primitives или Extended Primitives. Из появившегося меню выбрать желаемый объект;
  • выбрать в командной панели вкладку Create, в ней – вкладку Geometry. На этой вкладке выбрать набор примитивов Standard Primitives или Extended Primitives. После этого выбрать желаемый примитив.

Выбранный примитив можно построить двумя способами:

  • непосредственно в окне проекции с помощью мыши;
  • в командной панели.

Панель для построения примитива в командной панели содержит следующие основные элементы:

  • Name and Color – имя и цвет создаваемого примитива;
  • Creation Method – режим создания примитива (например, для параллелепипеда указывается режим Cube или Box);
  • Keyboard Entry – область для ввода параметров создаваемого примитива, например, для сферы - координаты центра и радиус, для параллелепипеда – координаты и размеры сторон, и т.д.;
  • Parameters – параметры объекта, используемые при его построении с помощью мыши в окне проекции.

Построение плоских фигур
Плоские фигуры (линия, прямоугольник, круг, звезда и т.д.) строятся аналогично объемным примитивам. Сначала требуется выбрать фигуру, которую требуется построить. Для этого выбрать элемент меню Create – Shapes, или в командной панели выбрать вкладку Create, затем – вкладку Shapes. После этого выбрать желаемую фигуру.
Затем построить её непосредственно в одном из окон проекций (с помощью мыши) или путем указания её параметров в командной панели.

4.2  Простейшие операции с объектами

4.2.1   Выделение объектов

Имеются два основных способа выделения объектов:

  • с помощью инструмента Select Object. С помощью этого инструмента можно выделять объекты щелчками мыши или построить рамку для выделения охватываемых объектов. Если используется рамка, то можно установить режим выделения с помощью кнопки Window Crossing: если она не нажата, то для выделения объекта достаточно выделить любую его часть, а если нажата, то для выделения объекта его нужно охватить рамкой полностью;
  • с помощью инструмента Select by Name. При нажатии этого инструмента появляется список имён всех имеющихся объектов. Из него можно выбрать один объект или несколько (используя клавиши Ctrl и Shift).

Если уже имеются выбранные объекты, то щелчок по объекту инструментом Select Object снимает выделение с объектов, выбранных ранее (вместо них выделяется новый объект). Чтобы добавлять объекты к выделенным (т.е. выделять новые объекты, не снимая выделение с выбранных ранее), требуется использовать инструмент Select Object при нажатой клавише Ctrl, а чтобы исключать объекты из числа выделенных – использовать Select Object при нажатой клавише Alt.

4.2.2   Изменение положения и размеров объектов. Удаление объектов

Перемещение объекта – инструмент Select and Move.
Поворот объекта – инструмент Select and Rotate.
Масштабирование объекта – инструмент Select and Scale. При этом можно выбрать режим масштабирования: равномерное (Uniform), неравномерное (Non-uniform), плавное (Squash).
Чтобы изменить параметры объекта, выделить его, затем в командной панели выбрать вкладку Modify и задать новые параметры.
Чтобы удалить объекты (один или несколько), выделить их и нажать клавишу Del. Можно также удалить все объекты сразу командой File – Reset.

4.2.3   Копирование и клонирование объектов

Чтобы выполнить копирование или клонирование объекта, требуется перемещать его с помощью инструмента Select and Move при нажатой клавише Shift. Появится окно управления копированием (см. рисунок 4.2).


Рисунок 4.2 – Окно управления копированием и клонированием
В этом окне выбирается режим копирования:

  • Copy – создаётся копия выбранного объекта, независимого от исходного;
  • Instance – создаются зависимые объекты (клоны). Изменение одного из них вызывает изменение другого (других);
  • Reference – создаётся ссылка на объект. Исходный и новый объект будут связаны друг с другом, но эта связь более сложная, чем для режима Instance. В этом случае изменение модификаторов, применённых до клонирования, отражается на всех объектах-клонах. В то же время применение новых модификаторов влияет только на один объект, но не на его клоны. Подробно модификаторы будут рассмотрены в разделе 4.3.

Копирование и клонирование объектов может выполняться также при их повороте и масштабировании, если выполнять их с помощью соответствующих инструментов при нажатой клавише Shift.
Ещё один способ копирования и клонирования – выбрать объект с помощью Select Object и выбрать операцию Edit – Clone (или нажать Ctrl-V). Появляется окно управления копированием, рассмотренное выше. Копия (или клон) создаётся совмещённой с исходным объектом. Чтобы работать с ними отдельно, удобно использовать выбор по имени (Select by Name).
Имеется также возможность создания зеркальных копий. Для этого выделить объект, затем нажать кнопку Mirror (или выбрать в меню Tools  команду Mirror). Появится окно управления отражением, показанное на рисунке 4.3.


Рисунок 4.3 – Окно управления зеркальным отражением
В этом окне требуется:

  • указать ось или плоскость, относительно которой требуется выполнить отражение;
  • выбрать режим отражения: No clone – выбранный объект отображается зеркально, оригинал не сохраняется; Copy, Instance или Reference – аналогично этим же режимам при копировании.

Все рассмотренные операции копирования, клонирования и отображения можно выполнять для нескольких объектов сразу. Для этого их требуется предварительно выбрать, как показано в п. 4.2.1.

4.2.4   Объединение объектов в группы

Чтобы объединить несколько объектов в группу, требуется выделить их и выбрать команду меню Group – Group. В появившемся окне указать имя создаваемой группы.
После этого все операции выделения, копирования, перемещения и т.д. будут применяться ко всем объектам группы.
Чтобы добавить объект в группу, требуется сначала выбрать объект, который нужно добавить. Затем выбрать команду Group – Attach и щёлкнуть мышью по одному из объектов в группе.
Чтобы исключить объект из группы, требуется выделить группу, затем выбрать команду Group – Detach и указать исключаемый объект.
Чтобы разгруппировать группу, требуется выделить ее, затем выбрать команду Group – Ungroup.
Группу можно также временно открыть (без разгруппирования) для работы с отдельными объектами. Для этого выделить группу и выбрать команду Group – Open. По окончании работы с отдельными объектами выполнить команду Group – Close.

4.2.5   Добавление объектов из другого файла

Пусть требуется добавить в файл один или несколько объектов (например, примитивов, плоских фигур или групп объектов) из другого файла.
Для этого сначала требуется загрузить файл, в который нужно добавить объекты. Затем выбрать команду File – Merge и указать файл, из которого нужно загрузить объекты. Появится окно со списком всех объектов выбранного файла. В этом окне выбрать желаемые объекты.

4.2.6   Сохранение изображения в растровом формате

Чтобы сохранить изображение в растровом формате (например, для использования в Photoshop), выбрать на экране окно с желаемым изображением (обычно – окно перспективы) и нажать Shift-Q. Появляется новое окно с выбранным изображением. Нажать кнопку Save Bitmap и выбрать желаемый формат.

4.2.7   Булевы операции

С объектами можно выполнять операции пересечения (Intersection), объединения (Union) или разности (Subtraction). Смысл этих операций аналогичен рассмотренному в п.3.6.3. Булевы операции имеют смысл только для пересекающихся объектов. Результатом такой операции является новый объект. Булевы операции выполняются следующим образом.
1    Выбрать в командной панели вкладку Create, в ней – вкладку Geometry. На этой вкладке выбрать команду Compound Object.
2    Выделить один из объектов, с которыми требуется выполнить булеву операцию (эти объекты должны пересекаться).
3    В командной панели нажать кнопку Boolean (она становится доступной только после выбора одного из объектов).
Примечание – Действия, соответствующие шагам 1-3, можно было выполнить и по-другому: выделить один из объектов, с которыми требуется выполнить операцию, и выбрать пункт меню Create – Compound Object – Boolean.
4    В командной панели (область Operation) указать выполняемую операцию: Union, Intersection, Substraction (A-B) или Substraction (B-A).
5    Щёлкнуть кнопку Pickup Operand B и выбрать второй объект для операции. Выбранная операция выполняется.
